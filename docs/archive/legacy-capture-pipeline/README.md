# Legacy Capture Pipeline Archive

This folder contains historical YoinkIt planning, calibration evidence, and
pre-Map-workbench design docs.

Archived here:

- `ROADMAP.md`: old execution roadmap and progress log.
- `SPEC.md`: pre-skill pipeline design of record.
- `PART-5-repair-loop-design.md`: repair-loop design rationale for the legacy
  capture pipeline.
- `calibration/`: historical calibration runs, scoreboards, reports, and raw
  evidence.

These files are useful for context and archaeology, but they are not the current
agent contract. Current agents should start from:

- `docs/ARCHITECTURE.md`
- `docs/CONTRACT.md`
- `CONTEXT.md`
- `docs/adr/`
